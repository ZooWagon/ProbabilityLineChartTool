## 概率折线图生成工具
Probability Line Chart Tool

时间：2022.2.6 15:10

### 功能说明

给定初始点数，增长阶段数，增长率符合的概率分布，

绘制点数变化的折线图，并将点数变化情况生成csv文件。

目前支持概率分布及其代号

0-正态分布
